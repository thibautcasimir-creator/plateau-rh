Comment publier vos modifications ?
1. Cliquez sur « Exporter HTML » → un fichier index.html se télécharge.
2. Dans GitHub, ouvrez votre dépôt → cliquez sur index.html → icône crayon (modifier).
3. Sélectionnez tout le contenu du fichier (Ctrl+A) → collez le contenu du fichier téléchargé.
4. Cliquez Commit changes. Le site se met à jour en ~2 minutes.
